# Платформа оценки городского благоустройства

## Краткий обзор
Это веб-приложение — интеллектуальная платформа для оценки состояния городской среды.  
Система повышает вовлечённость жителей в процессы благоустройства и предоставляет органам местного самоуправления **структурированные, актуальные данные в реальном времени**.

Пользователи могут:
- Отмечать проблемные зоны на карте (например, плохое освещение, сломанные лавочки, повреждённые дорожки)
- Оставлять текстовые отзывы и оценки
- Прикреплять фотографии  

---

## Предусмотрены роли пользователей
| Роль | Возможности |
|------|-------------|
| Незарегистрированный пользователь (гость) | Просмотр карты, объектов и отзывов |
| Зарегистрированный пользователь | Добавление отзывов и оценок, прикрепление фото, указание категории/тональности (Проблема/Предложение/Похвала), просмотр истории своих отзывов, управление личным кабинетом |
| Администратор | Управление справочниками (типы объектов, категории), модерация отзывов, модерация объектов, просмотр статистики |

---

## Основной функционал
- Просмотр и добавление объектов благоустройства  
- Оставление отзывов с оценкой и фотографиями  
- Категоризация и определение тональности отзывов  
- Администрирование и модерация данных  
- Статистические панели (планируется)  

---

## Цель проекта
Проект выполняется **в рамках производственной практики колледжа**.  

Задачи:
- Образовательные: применение знаний по веб-разработке и базам данных  
- Практические: создание реальной системы для оценки городской среды  
- Информационные: предоставление структурированных данных о состоянии объектов города для жителей и администрации

---

## Технологии
- **Фронтенд:** Vue.js  
- **Бэкенд:** Python (FastAPI)  
- **База данных:** PostgreSQL  
- **Дополнительно:** визуализация карт, аналитика и статистика  

---

## Примеры использования
> Пользователь обнаруживает проблему с освещением → отмечает объект на карте → оставляет отзыв с фото и категорией «Проблема» → данные доступны другим пользователям и администрации.
